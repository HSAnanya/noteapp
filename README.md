# 📝 React Notes App

A simple and efficient Notes application built using React. This app allows users to create, search, and delete notes dynamically.

---

## 🚀 Features

* ➕ Add new notes
* 🔍 Search notes instantly
* ❌ Delete notes
* ⚡ Fast and responsive UI
* ⚛️ Built using React Hooks (`useState`)

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5 & CSS3

## ⚙️ Installation & Setup

1. Clone the repository:

```id="zn0dlc"
git clone https://github.com/hsananya/noteapp.git
```

2. Navigate to the project:

```id="nfd0x3"
cd noteapp
```

3. Install dependencies:

```id="n3qzzm"
npm install
```

4. Run the app:

```id="ezdfy4"
npm start
```
